
# FancyChatBot

**FancyChatBot** is a Java Swing GUI Chat Bot with a beautiful and interactive interface.  

## Features
- User and Bot chat bubbles with different colors
- Dark theme with scrollable chat area
- Send messages using Enter key or Send button
- Simple AI responses in Bengali and English

## Installation & Running
1. Make sure **Java JDK** is installed (version 11 or higher).  
2. Clone or download this repository.  
3. Open terminal/command prompt in the project folder.  
4. Compile the Java code:
   ```bash
   javac FancyChatBot.java
